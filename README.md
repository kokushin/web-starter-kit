# web-starter-kit
gulp.js, EJS, webpack, PostCSS(cssnext), ESLint

## Download
[Source code (zip)](https://github.com/kokushin/web-starter-kit/releases/latest)

## install modules
```
$ yarn
```
or
```
$ npm install
```

## edit EJS config
```
src/ejs/_config.json
```

## start dev server
```
$ yarn dev
```
or
```
$ gulp
```

## build
```
$ yarn build
```
or
```
$ gulp build
```
